# Programme_for_guoyi
A  web crawler for three Websites 
三个中国某招标网站的爬虫，目标，实现：
1.抓取招标项目链接并且打开
2.翻页检索
3.关键词检索
4.项目内容相关信息检索
5.将特定信息写入Excel

运行环境：
Python3.6
PhantomJS

需安装的环境：
BeautifulSoup
Selenium
xlwt
PyQt5
